## 📌 Описание сервиса

BitOK предоставляет инструменты для:

* AML-проверок
* анализа криптоактивов
* мониторинга транзакций и адресов

Сервис используется для проверки кошельков и транзакций на наличие рискованных средств.

## 🔌 Работа с API

### Общие принципы

* API используется для получения данных по:

  * адресам
  * транзакциям
* взаимодействие происходит через HTTP-запросы
* доступ к методам предоставляется через Swagger

## 🧪 Использование Swagger

Swagger используется для:

* тестирования запросов
* просмотра доступных методов
* анализа ответов API

### Базовый процесс

1. Открыть Swagger
2. Выбрать нужный endpoint
3. Ввести параметры (адрес / транзакция)
4. Выполнить запрос
5. Проанализировать ответ

---

## 📊 Что возвращает API

В ответах API могут присутствовать:

* данные по адресу или транзакции
* оценка риска
* дополнительные параметры анализа

(конкретные поля смотреть в Swagger)
